[h1]自由生成物品。 搜尋你想要的物品和控制伺服器。 7 種不同的物品標籤方便使用。[/h1] [h1] 使用方法：[/h1] [list] [*] [h1] 普通： [/h1] [*] 按 T 鍵顯示選單。（可以在選項中更改） [*] CTRL + 點選選單中生命、精神、飢餓、溼度、溫度 和 伍迪的木頭值按鈕 將得到相反的結果。 [*] 搜尋步驟：點選 "搜尋你要的物品！" -> 輸入搜尋內容 -> Enter 鍵。 [*] 選擇生效玩家步驟：點選除錯選單中的 "人物" 指示按鈕 -> 輸入玩家的序號或暱稱 -> 確認。 [*] [h1] 物品生成：[/h1] [*] 單擊生成 1 個物品。（可以在選項中更改） [*] 右擊生成 10 個物品。（可以在選項中更改） [*] SHIFT + 單擊或右擊得到和物品生成數量份數的製作材料。 [*] CTRL + 點選加入或移除自定義物品列表。 [*] 自動 在你的物品欄中 或 人物旁邊 生成。 [*] [h1] 搜尋時的按鍵：[/h1] [*] Enter 鍵或點選任意位置：搜尋輸入的內容。 [*] UP 鍵：上個輸入內容。 [*] DOWN 鍵：下個輸入內容或清空輸入的內容。 [*] ESC 鍵：退出輸入。 [/list] [h1] 說明： [/h1] [list] [*] 客戶端MOD。 [*] 無需伺服器開啟，開了也沒用。 [*] 只有在檢測到擁有主機或管理員許可權時，MOD 的選單才會出現。 [*] 物品列表： All 所有, Foods 食物, Resources 資源, Weapons 武器, Tools 工具, Clothes 穿戴, Gifts 禮物, Livings 生物, buildings 建築和 Others 其他。 [*] 其他當前未新增入列表物品 和 MOD 物品在 Others 其他物品列表中。 [*] 僅支援在當前使用的物品列表裡搜尋。 [*] 搜尋功能會記錄搜尋過的內容。（最大記錄數量可以在選項中設定） [*] 其他功能：上帝模式 || 創造模式 || 一擊必殺模式 || 改變生命、精神、飢餓、溼度、溫度 和 伍迪的木頭刻度 || 改變 季節、時間、速度 和 天氣 || 殺人 || 復活 || 清空揹包 || 返回選人介面 || 召集隊友 || 重啟 || 換圖 || 回滾 || 儲存 || 地圖控制 || 傳送點記錄讀取 || 刪除、施肥、催熟、收穫、拾取、凍結 周圍目標 || 馴化牛 || 隨從控制 || 關掉伺服器 [*] 可以選擇命令對哪個玩家生效。（僅和你在同一個世界有效） [*] 本 MOD 的資料是可儲存的。（可以在選項中關閉） [*] 不同的世界的傳送功能資料會分開儲存。（放心的儲存你的位置吧！） [*] [b]不支援手柄或控制器。[/b] [/list] [h1] 注意：[/h1] [list] [*] 沒有這個 MOD 你也可以用命令達到相同的作用。它只是一個圖形化介面。 [*] 使用這個將會降低遊戲樂趣，請不要肆意使用。 [*] 其他物品列表裡的東西有可能會導致遊戲崩潰。(小心使用 ) [*] 如果喜歡這個 MOD，那麼點個贊吧！ [*] [url=http://steamcommunity.com/workshop/filedetails/discussion/551338671/490125737474973227/] 遊戲奔潰請看這裡。[/url] [/list] [h1] 如何新增管理員：[/h1] [list] [*] [h1] adminlist.txt：[/h1] [*] 請你的朋友加入你的遊戲。 [*] 在路徑：文件/klei/don't starve together/client_save(主機 ) 或 cluster.ini 同目錄(伺服器 ) 下建立 adminlist.txt 並開啟。 [*] 請你的朋友在遊戲中按 Y 或 U 鍵說句話。 [*] 在文字文件檔案：文件/klei/don't starve together/client_chat_log.txt 中查詢玩家的 ID。（示例：KU_********(線上) / OU_12345678901234567(離線)） [*] 複製 ID 到 adminlist.txt 並儲存。(每行一個 ID) [*] 重啟伺服器，就會發現添加了 ID 的朋友已是管理員。 [*] [h1] Steam 組：[/h1] [*] [url=http://forums.kleientertainment.com/topic/55994-server-admin-associate-your-server-with-a-steam-group/] 使用 Steam 組來管理伺服器及管理員。 [/url] [*] 設定 steam_group_admins = true，並且在 steam 組中設定玩家為官員（按兩次星，星變成黃金色）。 [*] 重啟伺服器，就會發現設定了官員的朋友已是管理員。 [/list] [h1] 其他：[/h1] [list] [*] [url=https://steamcommunity.com/workshop/filedetails/discussion/551338671/343787920139673533/] 歡迎參與翻譯 [/url] [*]
